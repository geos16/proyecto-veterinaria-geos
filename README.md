# Mi Aplicación [Veterinaria Geos]
Aplicación web full-stack con Node.js, Express y PostgreSQL.
## Funcionalidades
- Registro y login con JWT
- CRUD de mascotas con roles (user/admin)
- Vistas dinámicas con EJS
- Desplegada en Railway.com
## Tecnologías
- Node.js 24 LTS
- Express
- Sequelize
- JWT + bcrypt
- EJS
- express-validator, cookie-parser, SweetAlert2
## Enlace público (Render)
https://proyecto-final-geos-production.up.railway.app
## Instalación local
1. npm install
2. Crear .env con las variables
3. npm start
